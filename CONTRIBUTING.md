# Contributing to Media Tiles

Media Tiles is a community-friendly ecosystem for media artwork, Nuvio collection builds, and media tools.

At this stage, community build submissions are handled through a review-first workflow. Nothing is automatically published. This keeps the project organized, protects the repo, and helps make sure submitted builds are useful for other users.

---

## What Can Be Submitted?

The current build hub is focused on Nuvio-ready build packs that use AIOMetadata as the catalog layer.

Accepted submission types may include:

- Full Build
- Franchise Pack
- Genre Pack
- Decade Pack
- Network Pack
- Kids Pack
- Seasonal Pack
- Specialty Pack
- Other Nuvio-ready collection packs

For now, submitted builds should follow this standard:

```text
Platform: Nuvio
Catalog Layer: AIOMetadata
Import Style: Nuvio JSON + AIOMetadata JSON
